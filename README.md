# 共通テスト数学ⅠA ガチ完全再現版

理解はできるのに点が取り切れない人向けの、共通テスト形式トレーニングアプリ。
正誤判定だけでなく「解法の選び方」自体を訓練できる構成になっている。

## 構成ファイル

- `index.html` … 画面構造
- `style.css` … スタイル
- `app.js` … 状態管理・進行ロジック
- `questions.js` … 問題データ（第1問〜第4問、全21問構成）

## 主な機能

### 出題範囲
- 第1問：三角比・式変形（計算精度）
- 第2問：図形と計量（高さ・面積・方針切替）
- 第3問：内接円・接線の長さ・半径
- 第4問：誘導の読み方・時間配分

### 方針確認クイズ（route quiz）
解答する前に「この問題はどう解く？」と解法カテゴリ（三平方／角の二等分線／方べき／平方完成など）を選ばせる仕組み。
- 完全一致 → そのまま問題開始
- 部分一致（複数解法が必要な問題で一部だけ正解） → ヒントを出して継続
- 不一致 → 1回目は再選択を促し、2回目のミスで正解ルートを開示して開始

解答そのものの前に「解法を選ぶ」というワンステップを挟むことで、方針切替の弱さを可視化する。

### 解説（3段構成）
不正解・時間切れ・スキップ時いずれも以下を表示：
- ◆ 解き方（why）
- ◆ ミスしやすい点（mistake）
- ◆ 次へのコツ（tip）

### タイマー
- 設問ごとに異なる制限時間（`time`）
- 残り時間に応じて表示色が変化（通常→警告→危険）
- 「時間制限: 通常／厳格」を切替可能

### 弱点トラッカー
不正解時に以下4軸のいずれかへ加算：
- 計算精度／方針切替／時間判断／時間不足
最も多い軸を「最重要課題」として表示。

### ステージ別成績
第1問〜第4問ごとの正答率を個別表示。

### 復習モード
- 間違えた問題だけ再挑戦（`wrong` リスト）
- TIPSだけ復習（コツのみを一覧で確認、`tipList`）

### 履歴・保存
- 直近5回分の結果履歴（日時／正誤数／正答率／最重要課題／モード）を表示
- `localStorage` に状態を自動保存、再開可能
- 「学習ログだけリセット」「保存データ削除」をそれぞれ用意

### 休憩ガイド
10問到達時に一度休憩を促すメッセージを表示。

## データ仕様（questions.js）

各設問は以下のフィールドを持つ：

| フィールド | 内容 |
|---|---|
| `id` | 設問ID |
| `stage` / `num` | 出題ステージと番号 |
| `time` | 制限時間（秒） |
| `score` | 配点 |
| `weakness` | 不正解時に加算される弱点軸 |
| `route` | 正解の解法カテゴリ（配列、複数可） |
| `q` / `a` / `correct` | 問題文・選択肢・正解インデックス |
| `explain.why/mistake/tip` | 解き方・ミスパターン・コツ |

## 技術構成
- フレームワーク不使用（Vanilla JS）
- 状態管理はグローバル `state` / `stats` オブジェクト
- データ永続化は `localStorage`（キー：`kyotsu_app_v13`）

## 効果
- 答えの正誤だけでなく「どの解法に気づくべきか」を可視化し、初見の問題でも解法を選び取る力を養う
- 方針確認クイズにより、方針切替の弱さを早期に自覚できる
- mistake/tipにより、ミスパターンの認識と再発防止策を行動レベルに落とし込める
- 弱点トラッカー・ステージ別成績で、苦手分野を定量的に把握できる
